# Board43

Board43 is an RP2350A-based microcontroller development board. Flash PicoRuby or your favorite firmware and start hacking.

## Directory Structure

| Directory                     | Description                                                                                                                                                                                  | License                                          |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| [`docs/`](./docs)             | Reference manual and its compiler                                                                                                                                                            | MIT (compiler) / CC BY-SA 4.0 (document content) |
| [`firmware/`](./firmware)     | PicoRuby-based firmware                                                                                                                                                                      | MIT                                              |
| [`pcb/`](./pcb)               | KiCad PCB design data                                                                                                                                                                        | CERN-OHL-W-2.0 (some files under CC BY-SA 4.0)   |
| [`playground/`](./playground) | Browser-based PicoRuby IDE                                                                                                                                                                   | MIT                                              |
| [`workshop/`](./workshop)     | Workshop materials ([Japanese](https://github.com/smartbank-inc/Board43/blob/main/workshop/README.md) / [English](https://github.com/smartbank-inc/Board43/blob/main/workshop/README_en.md)) | CC BY-SA 4.0                                     |
